CryptoTracker

Knowledge is power

Cryptocurrencies are very volatile. It is important for investors to be aware of market trends in order to make the best decisions with their money. CryptoTracker is a website that provides live information on coin growth and volatility to keep users informed.

CryptoTracker utilizes CoinGecko's API to display live price information on the digital coins that are currently trending. Users can add coins into a portfolio and view them on the separate "pins" page. All pinned coins are saved through browser cookies. CryptoTracker suppports over 7000 coins, which can be searched for by the user to add to their portfolio. Components on the website are dynamically rendered using JavaScript and styled with CSS. Our group initially wanted to use a front-end framework such as ReactJS, but after prototyping we decided it was easier to avoid it given our project's smaller scope. However, the code is written in such a way that migrating to a different front-end will not take a lot of time.

Eric, Stefon, and Julian wrote the code for the website while Charles worked on the frequently asked questions and documentation. Everyone contributed to the technical writing in places like the readme, this presentation, and the original pitch. All 3 of the programmers contributed to the JavaScript and CSS components. Julian handled the graphs for the coins as well as the individual coin pages. Eric worked on the cookies and provided the layout for the website display. Aside from the bi-weekly meetings in class, our group met every Sunday to evaluate or progress and decide what features needed to be worked on. Communication between meetings was done through Discord. We utilized a GitHub repository to collaborate on code asyncronously and a shared document to assign tasks/deadlines.

Quality

Current cryptocurrency price information is scattered to say the least. Many sites are convoluted and don't refresh price information in real time. CryptoTracker provides a simple interface with live updates, making it a one-stop tracking site.

This product has a potential market of investors who want live information without the headache of manually keeping track of their portfolios. This website allows users to maintain a portfolio that automatically updates with current market data as well as a graph displaying some historical data. Having this much information consolidated into a single page better informs business decisions for users.

Moving forward, our website might have to migrate to a different API or create infrastructure that allows for aggregation of data on all coins. One of the issues with using CoinGecko is that API calls are limited to 50 coins at a time. This caused issues when calculating the top 10 most volatile coins, which led to us modifying the way coins are chosen for the home page. The current search feature on the website is not ideal, and would be one of the first improvements made given more time.
